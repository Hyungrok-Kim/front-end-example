하노이의 탑

하노이의 탑이란, 세 개의 기둥과 이 기둥에 꽂을 수 있는 크기가 다양한
원판들이 있고, 한 기둥의 원판들을 반대편의 기둥에 작은 것이 위에 있도록
순서대로 쌓는 퍼즐을 말합니다.

이 문제는 원판의 개수를 선정하여, 하노이의 탑의 최소 이동 경로를 구하고
해당 이동 경로에 대해 trace log(추적 기록)를 기록하는 문제입니다.
아래의 예시를 참조하여 hanoiStart() 함수를 완성해 보세요.
(컴퓨터의 사양을 고려하여 테스트는 최대 5 개까지만 가능하도록 조건을 설정해주세요.)

PS1 : 이 문제는 취업 코딩 테스트 문제로도 종종 출제 됩니다.
PS2 : 함수 내부에서 자기 자신을 호출하는 재귀함수의 개념을 활용합시다.

원판 개수 :  개  


결과 예시)
원판[1] : A -------> C
원판[2] : A -------> B
원판[1] : C -------> B
. . .
--------------
총 원판 : 3 개
최소 이동 횟수 : 7 번
